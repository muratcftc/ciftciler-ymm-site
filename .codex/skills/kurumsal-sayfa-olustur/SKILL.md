# Amaç
Çiftçiler YMM web sitesinde kullanılacak sayfaların yapısını, section kurgusunu ve metin iskeletini üretmek.

# Ne zaman kullanılır
- Ana sayfa oluşturulurken
- Hizmetler sayfası yazılırken
- Hakkımızda, iletişim veya teklif al sayfası tasarlanırken
- Mevcut kurumsal sayfa iyileştirilirken

# Öncelikler
1. Güven
2. Açıklık
3. Dönüşüm
4. Kurumsal görünüm
5. Okunabilirlik

# İçerik kuralları
- Dil varsayılan olarak Türkçe
- Kısa ve güçlü başlıklar kullan
- Paragrafları kısa tut
- Karmaşık vergi hizmetlerini sade anlat
- Aşırı reklam dili kullanma
- Dayanaksız başarı iddiaları yazma
- Uygun yerlerde teklif al veya iletişim çağrısı ekle

# Sayfa şablonu yaklaşımı
Mümkün olduğunda şu yapı tercih edilir:
1. Hero
2. Güven / uzmanlık vurgusu
3. Hizmet veya problem alanları
4. Yaklaşım / süreç bölümü
5. CTA alanı

# Tasarım yaklaşımı
- Sade premium kurumsal düzen
- Bol boşluk
- Net görsel hiyerarşi
- Ağır ama modern görünüm
- Koyu mavi / koyu petrol tonları + kırmızı vurgu

# Uyarılar
- Rastgele slogan üretme
- Fazla uzun hero metni yazma
- Hukuki kesinlik ifade eden cümleleri dikkatli kur
